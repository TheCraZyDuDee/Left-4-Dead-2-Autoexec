# Left 4 Dead 2 Autoexec

My own Personal Autoexec i worked on for the past Years.</br>
Each Category and Command should have their explanations if necessary and is easily customizable or expandable by executing your own configs.

## Usage
Download the Repository as zip [here](https://github.com/TheCraZyDuDee/Left-4-Dead-2-Autoexec/archive/refs/heads/main.zip).</br>
Move both the autoexec.cfg and the configs folder into the cfg folder of your game.</br>

Default Paths:
```
Windows: C:\Steam\steamapps\common\Left 4 Dead 2\left4dead2\cfg
Linux:   /home/user/.local/share/Steam/steamapps/common/Left 4 Dead 2/left4dead2/cfg
```
Recommended Launch Arguments can be found [here](https://github.com/TheCraZyDuDee/Left-4-Dead-2-Autoexec/blob/main/launch_arguments.md)<br>
Recommended Video Settings can be found [here](https://github.com/TheCraZyDuDee/Left-4-Dead-2-Autoexec/blob/main/video_settings.md)

Anything else should be self explanatory or explained in the config.
